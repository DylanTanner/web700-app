About This Website

This website serves as a demonstration of HTML5 principles and techniques, as well as routing in Express.

Here, you'll find three main sections:

  1. Home: This page.
  2. About: Learn more about me and my studies.
  3. HTML Demo: Explore various HTML elements and features.

Additionally, there are several routes that provide data, including:

  • All Students: View information about all students enrolled.
  • All Managers: Access details about all teaching assistants.
  • All Courses: See a list of all courses available.

Why HTML5?

This website utilizes HTML5, the latest version of the HTML standard, which offers improved support for multimedia, form input types, and semantic elements.

HTML5 enables us to create modern, interactive web experiences that are accessible across various devices and browsers.
